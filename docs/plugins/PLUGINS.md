## 플러그인 추가 요청

새로운 플러그인 번역을 요청하려면 GitHub Issue를 생성하세요:

**템플릿:**
```markdown
제목: [플러그인 번역 요청] 플러그인 이름

내용:
- 플러그인 ID:
- 플러그인 이름:
- 사용 이유:
- 다운로드 수 (선택):
- 참고 링크:
```

## 기여 방법

1. 번역하고 싶은 플러그인 선택
2. `version_updates/plugins/` 에 플러그인 폴더 생성
3. Properties 파일 추출 및 번역
4. 테스트 후 Pull Request
