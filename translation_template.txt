# ============================================================================
# 번역 작업 템플릿
# ============================================================================
# 이 파일은 누락된 번역 키들을 번역하기 위한 템플릿입니다.
# 
# 작업 방법:
# 1. 각 키의 원본 값(영어)을 확인합니다
# 2. 'TRANSLATE:' 뒤에 한국어 번역을 작성합니다
# 3. 번역 완료 후 apply_translations.py를 실행합니다
# 
# 주의사항:
# - {0}, {1} 같은 플레이스홀더는 그대로 유지하세요
# - _X 같은 단축키 표시는 한국어 뒤 괄호에 넣으세요 (예: 복사(_C))
# - ... (말줄임표)는 유지하세요
# - 특수문자는 백슬래시로 이스케이프하세요 (예: \:, \=)
# ============================================================================
